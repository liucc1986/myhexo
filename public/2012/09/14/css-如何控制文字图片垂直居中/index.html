文字图片垂直居中问题

固定高度容器中，单行文字垂直居中

设置行高

.box{
    height:50px;
    line-height:50px
}

固定容器高度，或者未知容器高度，多行文字垂直居中

.box{
        display: table-cell;
        width: 400px;
        height: 400px;
        vertical-align: middle;
        background:red;
        text-align: center;
    }
